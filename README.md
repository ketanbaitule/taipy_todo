# TODO App

![GitHub License](https://img.shields.io/github/license/ketanbaitule/taipy_todo)

A TODO App helps us to keep track our schedule.

## Features:

1. Add Task based on Type
2. Add Title
3. Delete Tasks

## Tech Stack Used

1. Taipy

## How To Install

```bash
pip install taipy
```

```
taipy run todo.py
```
